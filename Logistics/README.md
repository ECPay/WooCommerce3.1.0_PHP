# WooCommerce  綠界物流
---

## 1. 介紹

  - 綠界科技超商取貨付款外掛套件，提供合作特店以及個人會員使用開放原始碼商店系統時，無須自行處理複雜的檢核，直接透過安裝設定外掛套件，便可快速介接綠界科技物流整合系統，使用方便快速的商品運送機制。
    
  - 注意事項:若須同時使用金流模組，除了更新物流模組外，金流模組也請同步更新才能正常使用。


## 2. 適用版本
  - Wordpress Version 4.5.2 / 4.4.3 / 4.3.4 / 4.8.0
  - WooCommerce Version 2.5.5 / 2.4.13 / 3.0.9 / 3.1.1
