# WooCommerce APPLE PAY 金流模組
---

## 1. 介紹

  - 1. 需成為綠界的特約商店 (申請時間約10-15天) 
  - 2. APPLE僅接受銷售實體商品業者(非虛擬商品如遊戲點數) 
  - 3. 商店必須有APPLE的開發者帳號(個人/公司) (公司帳號申請時間約一個月,APPLE收年費為USD100) 
  - 4. APPLE PAY的付款交易須自備SSL加密環境 
  - 5. APPLE PAY交易環境必須支援Transport Layer Security (TLS) 1.2 


## 2. 安裝環境
  - WooCommerce  2.6.0 ~ 2.6.14 / 3.1.0 ~ 3.1.1
  - WordPress 4.5.3 / 4.6.1 / 4.7.3 / 4.8.0
